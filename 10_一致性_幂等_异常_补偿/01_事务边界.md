# 事务边界

## 本章结论

清结算平台内部状态变更使用本地事务；外部账户账务调用不与本地事务强绑定，必须通过入账单状态和 UNKNOWN 补偿处理不确定结果。

| 场景 | 事务边界 |
|---|---|
| 事件接入 | 写 SourceEvent 本地事务。 |
| 清分 | 写 ClearingResult/Item 与 SourceEvent 状态本地事务。 |
| 生成头寸 | 写 SettlementPosition 与 ClearingItem 状态本地事务。 |
| 确认结算 | 锁定头寸、生成单据、创建入账单本地事务。 |
| 调用账务 | 外部调用，结果回写本地事务。 |
