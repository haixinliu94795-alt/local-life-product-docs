# 页面：商品管理

## 1. 页面定位

商品管理页是本次商品下架/批量下架的核心后台操作入口。

## 2. 页面能力

| 能力 | 说明 |
| --- | --- |
| 状态筛选 | 支持全部、已上架、已下架、待审核、待上架等状态筛选。 |
| 单个操作 | 支持对单个商品进行上架、下架、编辑、删除等操作。 |
| 批量操作 | 支持批量上架、批量下架。 |
| 状态提示 | 根据商品状态展示不同操作能力。 |
| 限制提示 | 有未消费订单时，不允许删除或下架。 |

## 3. 关键交互

| 操作 | 交互 |
| --- | --- |
| 批量上架 | 二次确认；未审核或审核未通过商品统一提交审核，审核通过后自动上架。 |
| 批量下架 | 二次确认；有未消费订单商品自动过滤，仅下架无未消费订单商品。 |
| 删除商品 | 有未消费订单不可删除，删除商品不删除订单信息。 |
