# 关键提示文案

| 场景 | 文案 |
| --- | --- |
| 批量上架确认 | 确定要批量上架这些商品吗？ |
| 批量上架说明 | 批量上架商品时，所选商品中如有未审核或审核未通过的商品时，将统一提交审核。审核通过后自动上架。 |
| 批量下架确认 | 确定要批量下架这些商品吗？ |
| 批量下架说明 | 所选商品中如有未消费订单的商品，将自动过滤处理，仅可下架无未消费订单的商品。 |
| 单个下架确认 | 确定下架当前商品吗？ |
| 单个下架限制 | 如当前商品中如存在未消费订单时，将无法执行下架操作。 |
| 下架失败 | 当前商品中可能存在未消费订单，请核实后再次操作。 |
| 下架成功 | 下架成功！ |
| 上架成功 | 上架成功！ |
| 店铺已下架提示 | 谢谢光临，该店铺已下架 |
| 商品已下架提示 | 已下架 |

## 文案原则

1. 后台操作文案强调操作对象、限制条件和后果。
2. 用户端提示文案保持简洁，不展示内部运营原因。
3. 批量操作建议补充成功/失败数量，是否展示明细待确认。
