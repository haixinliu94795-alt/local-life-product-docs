# 页面：商品详情

## 1. 页面定位

商品详情页用于查看单个商品信息，并对单个商品执行上架/下架等操作。

## 2. 页面能力

| 能力 | 说明 |
| --- | --- |
| 商品信息查看 | 展示商品基本信息、购买须知、售卖信息等。 |
| 单个下架 | 支持对当前商品发起下架。 |
| 单个上架 | 支持对当前商品发起上架，必要时进入审核。 |
| 操作反馈 | 展示下架成功、下架失败、上架成功等反馈。 |

## 3. 关键规则

1. 如当前商品存在未消费订单，无法执行下架。
2. 下架失败时，需提示用户核实未消费订单后再操作。
3. 下架成功后，当前商品应进入已下架状态。
