## P0 定位

P0 只做本地生活普通商品核销后的清分、清算、结算、账务入账、后台商家应付和商户端展示。冻结/止付、自动出款、TRANSFER、完整对账差错、团餐/邀新/渠道全量接入均不进入 P0。
