# 本地生活清结算平台 DDD 技术设计 V001

本项目包用于沉淀“清结算平台”的一版技术设计。当前原型中的普通商品结算、商家应付、结算配置、结算记录、商户端财务展示等只作为一期需求输入；平台本身按成熟电商/本地生活清结算中心进行 DDD 领域建模。

## 文档定位

- **不是产品 PRD**：本包直接从原型/需求资料提取清结算相关需求，并进入技术设计。
- **不是只做结算记录**：本包按“清分 + 清算 + 结算 + 账务入账 + 对账差错”的完整清结算平台设计。
- **不是只服务本地生活**：一期先实现本地生活普通商品结算，后续可接团餐、邀新、渠道、按摩平台等业务。
