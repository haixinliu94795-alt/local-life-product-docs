# 领域事件

P0 只定义内部领域事件，不建设对外 outbox。

| 领域事件 | 触发时机 | 作用 |
|---|---|---|
| `SourceEventAccepted` | 来源事件入站成功 | 触发清分。 |
| `ClearingResultGenerated` | 清分完成 | 触发结算头寸生成。 |
| `SettlementPositionCreated` | 头寸生成 | 等待账期成熟。 |
| `SettlementPositionEligible` | 达到账期 | 可进入商家应付。 |
| `SettlementBillConfirmed` | 运营确认结算 | 触发账务入账。 |
| `AccountingPostingSucceeded` | 账务入账成功 | 结算单和头寸进入成功态。 |
| `AccountingPostingFailed` | 账务入账失败 | 结算单和头寸进入失败态。 |
| `AccountingPostingUnknown` | 账务调用结果未知 | 进入补偿查询。 |

退款、撤销、冲正领域事件不纳入 P0。
