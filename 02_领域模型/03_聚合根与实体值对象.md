# 聚合根与实体值对象

## 聚合根

| 聚合根 | 表 | 关键职责 |
|---|---|---|
| `SourceEvent` | `ccs_source_event` | 来源事件幂等、处理状态、事件载荷。 |
| `ClearingResult` | `ccs_clearing_result` | 订单/履约维度清分结果。 |
| `SettlementPosition` | `ccs_settlement_position` | 结算头寸生命周期和金额事实。 |
| `SettlementBill` | `ccs_settlement_bill` | 结算单状态机和账务入账驱动。 |
| `AccountingPostingOrder` | `ccs_accounting_posting_order` | 账务入账请求、重试、UNKNOWN 补偿。 |

## 实体

| 实体 | 所属聚合 | 说明 |
|---|---|---|
| `ClearingResultItem` | `ClearingResult` | 分参与方、分金额类型的清分明细。 |
| `SettlementBillItem` | `SettlementBill` | 结算单包含的头寸明细。 |
| `SettlementOperationLog` | 独立追加日志 | 操作审计。 |

## 值对象

| 值对象 | 说明 |
|---|---|
| `MoneyCent` | 人民币分，非负金额；不设计币种字段。 |
| `BusinessIdentity` | `caller_system + biz_domain + biz_no`。 |
| `Idempotency` | `idempotent_key + request_hash`。 |
| `SettlementTarget` | 结算对象类型和结算对象编号。P0 为商户。 |
| `AccountingReference` | `accounting_request_no + fund_account_flow_no`。 |
