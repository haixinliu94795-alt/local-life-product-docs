# 领域模型总览

## 1. 限界上下文

```text
Settlement Intake Context      来源事件接入
Clearing Context               清分
Settlement Position Context    清算/结算头寸
Settlement Billing Context     结算单
Accounting Posting Context     账务入账编排
Settlement Query Context       标准查询投影
Operation Audit Context        操作审计
```

## 2. 领域对象关系

```text
SourceEvent
  -> ClearingResult
      -> ClearingResultItem
          -> SettlementPosition
              -> SettlementBillItem
                  -> SettlementBill
                      -> AccountingPostingOrder
```

## 3. 主关联对象

| 项 | 值 |
|---|---|
| 主聚合 | `SettlementBill` |
| 主关联编号 | `bill_no` |
| 业务接入编号 | `event_no` |
| 清分编号 | `clearing_no` / `clearing_item_no` |
| 结算头寸编号 | `position_no` |
| 账务入账编号 | `posting_no` |
| 账务平台回写编号 | `accounting_request_no` / `fund_account_flow_no` |
