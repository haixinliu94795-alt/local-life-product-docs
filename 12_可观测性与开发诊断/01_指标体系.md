# 指标体系

## 本章结论

指标用于开发联调、测试验收和后续运维接入，不涉及上线部署方案。

| 指标 | 说明 |
|---|---|
| source_event_received_total | 接入事件数。 |
| clearing_success_total | 清分成功数。 |
| position_eligible_total | 到期可结算头寸数。 |
| settlement_bill_created_total | 结算单生成数。 |
| accounting_posting_unknown_total | 账务 UNKNOWN 数。 |
| idempotent_conflict_total | 幂等冲突数。 |
