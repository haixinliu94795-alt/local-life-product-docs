# 部署与开关策略

## 1. 开关原则

- 开关属于接入层或配置平台，不属于清结算领域模型。
- 不在 DDL 中增加 `cutover_time` 字段。
- 不用旧系统字段驱动新平台状态。

## 2. 推荐开关

| 开关 | 说明 |
|---|---|
| `ccs.source-event.enabled` | 是否接收来源事件。 |
| `ccs.business-scene.local-life-goods.enabled` | 是否启用本地生活普通商品场景。 |
| `ccs.accounting-posting.enabled` | 是否调用账务平台。 |
| `ccs.accounting-compensation.enabled` | 是否启用 UNKNOWN 查询补偿。 |
