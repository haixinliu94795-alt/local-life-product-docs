# 错误码与返回码

| 错误码 | 说明 | HTTP 建议 |
|---|---|---:|
| `CCS_IDEMPOTENT_CONFLICT` | 同幂等键请求参数不一致 | 409 |
| `CCS_PENDING_ITEM_NOT_FOUND` | 待结算项不存在 | 404 |
| `CCS_PENDING_ITEM_NOT_ELIGIBLE` | 待结算项不是可结算状态 | 400 |
| `CCS_PENDING_ITEM_LOCKED` | 待结算项已被锁定 | 409 |
| `CCS_MERCHANT_NOT_SAME` | 批量结算非同一商户 | 400 |
| `CCS_SUBJECT_NOT_SAME` | 批量结算非同一结算主体 | 400 |
| `CCS_AMOUNT_NOT_MATCH` | 汇总金额与明细金额不一致 | 400 |
| `CCS_BILL_STATUS_INVALID` | 结算单状态不允许当前操作 | 409 |
| `CCS_ACCOUNTING_UNKNOWN` | 账务结果未知 | 202 |
| `CCS_ACCOUNTING_FAILED` | 账务明确失败 | 500 |
| `CCS_LEGACY_CUTOFF_REQUIRED` | 缺少新老切换时间配置 | 500 |
