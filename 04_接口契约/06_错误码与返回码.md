# 错误码与返回码

| 错误码 | HTTP | 说明 |
|---|---:|---|
| `CCS_IDEMPOTENT_CONFLICT` | 409 | 同一幂等键请求指纹不一致。 |
| `CCS_SOURCE_EVENT_DUPLICATED` | 200 | 来源事件重复，返回原处理结果。 |
| `CCS_POSITION_NOT_ELIGIBLE` | 400 | 头寸不是可结算状态。 |
| `CCS_POSITION_LOCKED` | 409 | 头寸已被其他结算单锁定。 |
| `CCS_POSITION_MERCHANT_MISMATCH` | 400 | 批量头寸不属于同一商户。 |
| `CCS_POSITION_SCENE_MISMATCH` | 400 | 批量头寸不属于同一业务场景。 |
| `CCS_BILL_STATUS_ILLEGAL` | 400 | 结算单状态不允许当前动作。 |
| `CCS_ACCOUNTING_FAILED` | 500 | 账务平台明确失败。 |
| `CCS_ACCOUNTING_UNKNOWN` | 202 | 账务平台结果未知，需要补偿查询。 |
