# 清算与商家应付接口

## 1. 查询可结算头寸

```text
GET /admin/ccs/settlement-positions/eligible
```

查询条件：

| 字段 | 说明 |
|---|---|
| `merchantNo` | 商户编号。 |
| `businessScene` | 业务场景。 |
| `eligibleTimeStart` / `eligibleTimeEnd` | 可结算时间范围。 |
| `sourceOrderNo` | 来源订单号。 |
| `sourceFulfillmentNo` | 来源履约号。 |
| `pageNo` / `pageSize` | 分页。 |

返回字段以 `SettlementPositionVO` 为准，只返回 `ELIGIBLE` 状态头寸。

## 2. 导出可结算头寸

导出属于查询能力，不进入结算写流程。

```text
GET /admin/ccs/settlement-positions/eligible/export
```
