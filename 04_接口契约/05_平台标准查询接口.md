# 平台标准查询接口

本文件定义清结算平台底层标准查询能力。它不是商户端页面接口，也不承诺“待结算”等产品展示口径。

## 1. 结算头寸查询

```text
GET /ccs/settlement-positions
```

支持按状态查询：

```text
IN_TRANSIT / ELIGIBLE / LOCKED / ACCOUNTING_PROCESSING / ACCOUNTING_FAILED / UNKNOWN / ACCOUNTED
```

## 2. 结算单查询

```text
GET /ccs/settlement-bills
GET /ccs/settlement-bills/{billNo}
```

## 3. 账务入账单查询

```text
GET /ccs/accounting-posting-orders
GET /ccs/accounting-posting-orders/{postingNo}
```

上层展示系统可以消费这些标准查询能力，但展示聚合规则不属于清结算平台 P0。
