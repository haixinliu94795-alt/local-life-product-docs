# 账务平台对接契约

## 1. P0 调用接口

P0 不新增 `recordGenericSettlement`，先复用当前已有：

```java
AccountingFacade.recordProductSettlement(...)
```

## 2. 调用字段

| 字段 | 取值 |
|---|---|
| `callerSystem` | `CLEARING_SETTLEMENT_PLATFORM` |
| `bizDomain` | `LOCAL_LIFE` |
| `bizNo` | `bill_no` |
| `accountingScene` | `PRODUCT_SETTLEMENT` |
| `idempotentKey` | `CCS:PRODUCT_SETTLEMENT:{bill_no}:ACCOUNTING` |
| `amount` | `ccs_settlement_bill.bill_amount_cent` |
| `merchantId` | `merchant_id` |

## 3. 回写字段

- `accounting_request_no`
- `fund_account_flow_no`
- `accounting_success_time`

如果当前账务接口返回值不包含上述字段，开发前必须确认是否通过查询接口补回。
