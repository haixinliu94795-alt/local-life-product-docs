# 状态机矩阵测试

## 必测

1. `IN_TRANSIT -> ELIGIBLE`。
2. `ELIGIBLE -> LOCKED`。
3. `LOCKED -> ACCOUNTING_PROCESSING`。
4. `ACCOUNTING_PROCESSING -> ACCOUNTED`。
5. `ACCOUNTING_PROCESSING -> ACCOUNTING_FAILED`。
6. `ACCOUNTING_PROCESSING -> UNKNOWN`。
7. `ACCOUNTING_FAILED -> ACCOUNTING_PROCESSING` 重试。
8. `UNKNOWN -> ACCOUNTED` 查询回正。
9. `UNKNOWN -> ACCOUNTING_FAILED` 查询回正。
10. `CREATED/CONFIRMED -> CANCELED` 入账前取消。

## 禁止流转测试

1. `ACCOUNTED` 不可回退。
2. `ACCOUNTING_PROCESSING` 不可取消。
3. `UNKNOWN` 不可直接重复入账。
4. 非 `ELIGIBLE` 头寸不可被结算确认。
