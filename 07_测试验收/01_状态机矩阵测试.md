# 状态机矩阵测试

测试必须从 `03_流程与状态机` 中的状态矩阵生成，至少覆盖：

| 用例 | 目标 |
|---|---|
| `IN_TRANSIT -> ELIGIBLE` | 账期成熟。 |
| `ELIGIBLE -> LOCKED` | 确认结算锁定。 |
| `LOCKED -> ACCOUNTING_PROCESSING` | 开始入账。 |
| `ACCOUNTING_PROCESSING -> ACCOUNTED` | 账务成功。 |
| `ACCOUNTING_PROCESSING -> ACCOUNTING_FAILED` | 账务失败。 |
| `ACCOUNTING_PROCESSING -> UNKNOWN` | 账务未知。 |
| `ACCOUNTING_FAILED -> ACCOUNTING_PROCESSING` | 重试。 |
| `CREATED -> CANCELED` | 入账前取消。 |
| `ACCOUNTED -> 非终态` | 禁止。 |
| `UNKNOWN -> 直接重试` | 禁止。 |
