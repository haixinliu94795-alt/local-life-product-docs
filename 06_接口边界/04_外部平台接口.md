# 外部平台接口

## 1. 账户账务平台

| 能力 | 一期处理 |
|---|---|
| 普通商品结算入账 | 复用现有 `recordProductSettlement`。 |
| 服务订单结算入账 | 后续服务场景复用 `recordServiceOrderSettlement`。 |
| 通用结算入账 | 建议 P1 新增 `recordGenericSettlement`，避免业务类型硬编码。 |
| 幂等查询 | 建议补充按 `idempotent_key` 查询账务请求结果。 |

## 2. 出款/提现平台

| 能力 | 清结算平台关系 |
|---|---|
| 商户提现申请 | 由出款平台处理。 |
| 提现中金额 | 商户端聚合展示。 |
| 提现成功金额 | 商户端聚合展示。 |
| 出款失败 | 不回滚结算单状态，但会影响提现状态。 |

## 3. 支付平台

支付平台提供支付成功、退款成功、渠道账单和支付对账结果。清结算平台不直接处理收单，只消费支付/退款事实。
