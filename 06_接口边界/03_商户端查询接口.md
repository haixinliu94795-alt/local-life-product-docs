# 商户端查询接口

| 接口 | 说明 | 数据来源 |
|---|---|---|
| `GET /merchant/clear-settlement/summary` | 商户资金摘要：待结算、已结算、已提现、提现中。 | 清结算平台 + 账户账务 + 出款平台 |
| `GET /merchant/clear-settlement/pending-items` | 待结算明细。 | cs_pending_settlement_item 投影 |
| `GET /merchant/clear-settlement/bills` | 结算记录。 | cs_settlement_bill 投影 |
| `GET /merchant/clear-settlement/bills/{billNo}` | 结算详情。 | bill + bill_item |
| `GET /merchant/account/statements` | 账户明细。 | 账户账务平台，清结算补充单据信息 |

## 金额口径

| 展示项 | 口径 |
|---|---|
| 待结算金额 | 待结算池中未结算、未取消、未冻结或按展示规则计入的金额。 |
| 已结算金额 | `ACCOUNTING_SUCCESS` 结算单汇总。 |
| 可提现余额 | 账户账务平台实时余额。 |
| 已提现/提现中 | 出款平台统计。 |

已结算金额不等于可提现余额：可提现余额还会受到提现中冻结、已提现、退款冲正、账户调整等因素影响。
