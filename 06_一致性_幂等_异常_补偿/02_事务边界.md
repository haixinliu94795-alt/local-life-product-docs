# 事务边界

## 1. 来源事件事务

同一事务内完成：

1. 写入 `ccs_source_event`；
2. 生成 `ccs_clearing_result`；
3. 生成 `ccs_clearing_result_item`；
4. 生成 `ccs_settlement_position`。

## 2. 结算确认事务

同一事务内完成：

1. 校验头寸均为 `ELIGIBLE`；
2. 乐观锁更新头寸为 `LOCKED`；
3. 创建 `ccs_settlement_batch`；
4. 创建 `ccs_settlement_bill`；
5. 创建 `ccs_settlement_bill_item`；
6. 写操作日志。

## 3. 账务入账事务

本地创建账务入账单和外部账务调用分阶段执行：

1. 本地事务创建 `ccs_accounting_posting_order` 并推进结算单状态；
2. 调用账户账务平台；
3. 根据结果回写本地状态；
4. 调用结果未知时进入 `UNKNOWN`，后续补偿查询。
