# 幂等规则

## 1. 来源事件幂等

```text
idempotent_key = CCS:SOURCE:{callerSystem}:{bizDomain}:{bizNo}
```

## 2. 结算确认幂等

```text
idempotent_key = CCS:SETTLEMENT_CONFIRM:{callerSystem}:{bizDomain}:{bizNo}
```

## 3. 账务入账幂等

```text
accounting_idempotent_key = CCS:ACCOUNTING_POSTING:{billNo}
```

账务失败重试和 UNKNOWN 查询都必须复用同一账务幂等键。
