# 统一响应与错误结构

## 响应结构

```json
{
  "success": true,
  "code": "SUCCESS",
  "message": "success",
  "data": {}
}
```

## 错误结构

```json
{
  "success": false,
  "code": "CCS_IDEMPOTENT_CONFLICT",
  "message": "同一幂等键下请求参数不一致",
  "data": null
}
```

## 关键错误码

| 错误码 | 含义 |
|---|---|
| `CCS_IDEMPOTENT_CONFLICT` | 幂等键相同但 request_hash 不一致。 |
| `CCS_POSITION_NOT_ELIGIBLE` | 头寸不可结算。 |
| `CCS_BILL_AMOUNT_MISMATCH` | 结算单金额与明细不一致。 |
| `CCS_ACCOUNTING_UNKNOWN` | 账务结果未知。 |
| `CCS_POLICY_NOT_FOUND` | 未找到有效策略。 |
