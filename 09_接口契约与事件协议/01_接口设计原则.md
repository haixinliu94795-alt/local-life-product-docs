# 接口设计原则

## 本章结论

接口按语义分为 Command、Query、Event、Port 四类。写入接口必须幂等，查询接口不得产生副作用，外部平台调用必须通过防腐端口。

| 类型 | 说明 | 示例 |
|---|---|---|
| Event | 业务事实输入 | `acceptSourceEvent` |
| Command | 改变清结算状态 | `confirmMerchantSettlement` |
| Query | 查询标准领域数据 | `querySettlementBill` |
| Port | 对接外部平台 | `SettlementAccountingPort` |
