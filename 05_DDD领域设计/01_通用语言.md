# 通用语言

## 本章结论

清结算平台采用成熟行业领域语言：来源事件、清分、清算、结算头寸、结算单、账务入账单、策略快照。任何旧系统字段或页面命名都不能替代通用语言。

| 术语 | 英文 | 定义 |
|---|---|---|
| 来源事件 | SourceEvent | 业务平台传入的标准业务事实。 |
| 清分结果 | ClearingResult | 对一笔业务事实计算出的资金归属结果。 |
| 清分明细 | ClearingResultItem | 参与方、金额项、方向和金额。 |
| 结算头寸 | SettlementPosition | 某个清分金额项在结算前后的生命周期载体。 |
| 结算批次 | SettlementBatch | 一次批量结算操作的批次。 |
| 结算单 | SettlementBill | 面向结算主体的一张结算单据。 |
| 账务入账单 | AccountingPostingOrder | 调用账户账务平台的入账编排单。 |
| 策略快照 | PolicySnapshot | 清分和账期计算时命中的规则快照。 |
