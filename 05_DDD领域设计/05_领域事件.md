# 领域事件

## 本章结论

P0 领域事件主要用于内部解耦和审计，不要求建设对外 Outbox。事件状态仍以主表状态机为准。

| 领域事件 | 触发时机 | 消费方 |
|---|---|---|
| `SourceEventAccepted` | 来源事件接收成功 | 清分应用服务 |
| `ClearingResultGenerated` | 清分成功 | 头寸应用服务 |
| `SettlementPositionEligible` | 账期成熟 | 商家应付查询/运营确认 |
| `SettlementBillConfirmed` | 结算确认 | 入账编排服务 |
| `AccountingPostingSucceeded` | 入账成功 | 结算单状态联动 |
| `AccountingPostingUnknown` | 入账结果未知 | 补偿任务/人工诊断 |
