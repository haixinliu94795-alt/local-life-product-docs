# 主链路总览

```text
业务/履约平台
  -> FULFILLMENT_CONFIRMED 标准事件
  -> 来源事件接入
  -> 清分结果 ClearingResult
  -> 清分明细 ClearingResultItem
  -> 结算头寸 SettlementPosition(IN_TRANSIT)
  -> 账期成熟 SettlementPosition(ELIGIBLE)
  -> 后台商家应付查询
  -> confirmMerchantSettlement
  -> SettlementBatch / SettlementBill / SettlementBillItem
  -> AccountingPostingOrder
  -> 账户账务平台入账
  -> ACCOUNTED
```

P0 不处理退款、提现、冻结、旧链路切换和页面展示口径。
