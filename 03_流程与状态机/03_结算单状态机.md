# 结算单状态机

## 1. 状态定义

| 状态 | 说明 | 是否可取消 | 是否可重试 | 是否终态 |
|---|---|---:|---:|---:|
| `CREATED` | 结算单已创建。 | 是 | 否 | 否 |
| `CONFIRMED` | 结算确认完成，准备入账。 | 是 | 是 | 否 |
| `ACCOUNTING_PROCESSING` | 账务入账中。 | 否 | 否 | 否 |
| `ACCOUNTED` | 入账成功。 | 否 | 否 | 是 |
| `ACCOUNTING_FAILED` | 入账失败。 | 否 | 是 | 否 |
| `UNKNOWN` | 账务结果未知。 | 否 | 需先查询 | 否 |
| `CANCELED` | 入账前取消。 | 否 | 否 | 是 |

## 2. 状态流转矩阵

| 当前状态 | 触发事件 | 前置条件 | 领域动作 | 目标状态 | 失败目标状态 | 是否幂等 | 是否可重试 | 测试用例 |
|---|---|---|---|---|---|---|---|---|
| - | createBill | 头寸锁定成功 | 创建结算单 | `CREATED` | - | 是 | 是 | TC-BILL-001 |
| `CREATED` | confirm | 操作人确认 | 记录确认信息 | `CONFIRMED` | `CREATED` | 是 | 是 | TC-BILL-002 |
| `CONFIRMED` | startAccounting | 账务入账单创建 | 调用账务平台 | `ACCOUNTING_PROCESSING` | `ACCOUNTING_FAILED` | 是 | 是 | TC-BILL-003 |
| `ACCOUNTING_PROCESSING` | accountingSuccess | 账务明确成功 | 回写账务编号 | `ACCOUNTED` | `UNKNOWN` | 是 | 是 | TC-BILL-004 |
| `ACCOUNTING_PROCESSING` | accountingFailed | 账务明确失败 | 记录失败原因 | `ACCOUNTING_FAILED` | `UNKNOWN` | 是 | 是 | TC-BILL-005 |
| `ACCOUNTING_PROCESSING` | accountingUnknown | 超时或结果不明 | 标记未知 | `UNKNOWN` | `UNKNOWN` | 是 | 是 | TC-BILL-006 |
| `ACCOUNTING_FAILED` | retryAccounting | 使用同一账务幂等键 | 再次调用账务 | `ACCOUNTING_PROCESSING` | `ACCOUNTING_FAILED` | 是 | 是 | TC-BILL-007 |
| `UNKNOWN` | queryAccountingSuccess | 查询到账务成功 | 回正成功 | `ACCOUNTED` | `UNKNOWN` | 是 | 是 | TC-BILL-008 |
| `UNKNOWN` | queryAccountingFailed | 查询到账务失败 | 回正失败 | `ACCOUNTING_FAILED` | `UNKNOWN` | 是 | 是 | TC-BILL-009 |
| `CREATED` | cancel | 未发起账务 | 释放头寸 | `CANCELED` | `CREATED` | 是 | 否 | TC-BILL-010 |
| `CONFIRMED` | cancelBeforeAccounting | 未发起账务 | 释放头寸 | `CANCELED` | `CONFIRMED` | 是 | 否 | TC-BILL-011 |

## 3. 幂等行为

重复确认结算时：

| 场景 | 处理 |
|---|---|
| `idempotent_key` 相同且 `request_hash` 相同 | 返回原 `bill_no`。 |
| `idempotent_key` 相同但 `request_hash` 不同 | 拒绝，错误码 `CCS_IDEMPOTENT_CONFLICT`。 |
| 头寸已被其他结算单锁定 | 拒绝，错误码 `CCS_POSITION_LOCKED`。 |
