# 账务入账流程

## 本章结论

账务入账是外部平台协作，必须通过 `SettlementAccountingPort` 防腐层，不允许直接在领域模型中绑定具体账户账务实现。

## 流程

```text
AccountingPostingOrder.INIT
  -> REQUESTING
  -> 调用 SettlementAccountingPort.postSettlement
  -> SUCCESS / FAILED / UNKNOWN
  -> 联动 SettlementBill 和 SettlementPosition
```

## UNKNOWN 处理

如果账务调用超时或结果不可判定，入账单进入 `UNKNOWN`。此时不允许直接重入账，必须先通过幂等键或业务键查询账务平台结果。
