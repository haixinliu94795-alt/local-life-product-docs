# 开发总说明

## 1. 推荐工程模块

```text
local-live-module-clearing-settlement
```

如短期不能新增 Maven 模块，可先放入：

```text
local-live-module-service/src/main/java/org/jeecg/modules/ccs
```

但包名必须使用 `ccs` 或 `clearingSettlement`，不得继续放在 `localSettlementRecord` 领域包下。

## 2. 分层包结构

```text
org.jeecg.modules.ccs
  adapter
    accounting
    legacy
  application
    command
    query
  domain
    clearing
    settlement
    accounting
  infrastructure
    mapper
    repository
  interfaces
    admin
    internal
    merchant
```

## 3. 禁止项

- 不得用旧 `local_settlement_record` 作为新平台主表。
- 不得新增 `currency / cur / currency_code / tenant_id` 字段。
- 不得绕过状态机直接更新状态。
- 不得在 `UNKNOWN` 状态直接重复入账。
- 不得把出款核心 TRANSFER 接入 P0 主链路。
