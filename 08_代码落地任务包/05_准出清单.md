# 准出清单

| 检查项 | 必须满足 |
|---|---|
| DDL | 所有表可执行；无 currency、tenant_id、DECIMAL 金额。 |
| Entity | 字段与 DDL 一致；金额 Long；状态枚举一致。 |
| 状态机 | 所有允许/禁止流转有代码判断和测试。 |
| 幂等 | 同键同 hash 返回原结果；同键异 hash 拒绝。 |
| 账务 | 成功、失败、UNKNOWN、重试均可测。 |
| 退款 | 结算前/中/后最小规则可测。 |
| 双轨 | 新老 BFF 聚合可测。 |
| 日志 | 操作日志、traceId、失败原因可追踪。 |
| 禁区 | 不修改旧链路主逻辑，不接 TRANSFER，不做自动出款。 |
