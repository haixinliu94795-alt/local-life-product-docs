# 成本承担与分佣模型

## 本章结论

平台优惠、商家优惠、会员折扣、金豆抵扣、推广分佣、邀新分佣都应统一进入清分金额项，而不是散落在结算单字段中。

| 业务配置 | 清分金额项 |
|---|---|
| 平台优惠券成本 | `PLATFORM_DISCOUNT_COST` |
| 商户优惠成本 | `MERCHANT_DISCOUNT_COST` |
| 会员折扣成本 | `MEMBER_DISCOUNT_COST` |
| 金豆抵扣成本 | `GOLD_BEAN_COST` |
| 推广分佣 | `PROMOTION_COMMISSION` |
| 邀新分佣 | `INVITE_COMMISSION` |
