# 账期策略模型

## 本章结论

账期策略的输出不是“待结算”，而是 `eligible_time` 和结算归属周期。头寸在 `eligible_time` 之前为 `IN_TRANSIT`，到期后为 `ELIGIBLE`。

| 字段 | 说明 |
|---|---|
| `settlement_cycle_type` | T_PLUS_N / PERIODIC / MANUAL。 |
| `cycle_basis` | WRITE_OFF_TIME / FULFILLMENT_TIME / PAY_TIME。 |
| `cycle_days` | T+N 天数。 |
| `settlement_period_start_time` | 结算归属周期开始。 |
| `settlement_period_end_time` | 结算归属周期结束。 |
| `eligible_time` | 可结算时间。 |
