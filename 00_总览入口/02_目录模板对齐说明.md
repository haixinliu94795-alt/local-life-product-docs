# 目录模板对齐说明

V004 已按正式技术方案目录规范调整为九大 active 目录：

| 目录 | 作用 |
|---|---|
| `01_业务场景与需求` | 一期范围、非范围、产品口径与底层口径适配。 |
| `02_领域模型` | DDD 通用语言、限界上下文、聚合、不变量、领域事件。 |
| `03_流程与状态机` | 主链路、异常链路、状态矩阵和状态联动。 |
| `04_接口契约` | 事件接入、后台、商户端 BFF、账务对接和错误码。 |
| `05_数据模型` | 表设计、DDL、索引、幂等约束、旧表映射。 |
| `06_一致性_幂等_异常_补偿` | 事务边界、幂等、账务失败/UNKNOWN、退款三场景、诊断能力。 |
| `07_测试验收` | 状态机、幂等、异常补偿、退款和双轨回归。 |
| `08_代码落地任务包` | Codex 任务卡、代码结构、DTO 契约和准出清单。 |
| `09_上线与运维` | 灰度切换、上线后验收、回滚预案。 |

`99_参考输入` 仅供追溯，不作为 active 实现目录。
