# 开发准出清单

## 1. 必须通过

| 项 | 准出标准 |
|---|---|
| DDL | 与 `03_DDL_V010_P0.sql` 一致。 |
| request_hash | 三类写请求 hash 单测通过。 |
| SourceEvent | 幂等接入和冲突拒绝通过。 |
| ClearingResult | 清分金额校验通过。 |
| MaturityJob | `IN_TRANSIT -> ELIGIBLE` 测试通过。 |
| ConfirmSettlement | 单条、批量、超限、混入不可结算项测试通过。 |
| OperationLog | 日志失败不回滚主事务。 |
| AccountingPosting | SUCCESS/FAILED/UNKNOWN 状态联动通过。 |
| UNKNOWN 补偿 | SLA 规则测试通过。 |
| OpenAPI | 与实现 DTO 字段一致。 |

## 2. 禁止准出

出现以下任一情况不得准出：

```text
存在冻结/止付 P0 状态
存在退款 P0 主链路
存在出款 P0 主链路
存在 /eligible/export P0 接口
存在 UNKNOWN 直接重入账
存在 DECIMAL 金额字段
存在 currency / tenant_id 字段
```
