# 类职责清单

| 类 | 职责 |
|---|---|
| `SourceEventApplicationService` | 接收事件、幂等、触发清分。 |
| `ClearingApplicationService` | 编排清分。 |
| `SettlementPositionApplicationService` | 生成头寸、推进账期成熟。 |
| `SettlementBillingApplicationService` | 确认结算、生成批次和结算单。 |
| `AccountingPostingApplicationService` | 调用账务、UNKNOWN 补偿。 |
| `SettlementAccountingPort` | 账户账务平台防腐端口。 |

## V009 新增类职责

| 类 | 职责 |
|---|---|
| `PolicyBindingApplicationService` | 管理策略绑定创建、禁用、查询。 |
| `PolicyMatchDomainService` | 按业务上下文匹配生效策略。 |
| `SettlementPolicyVersionFactory` | 创建带结算模式、发起方式、阈值、留存字段的策略版本。 |
| `SettlementAmountPolicyEvaluator` | 计算结算单留存金额和本次入账金额；P0 默认留存 0。 |
| `PolicySnapshotService` | 生成清分和结算时使用的策略快照。 |
