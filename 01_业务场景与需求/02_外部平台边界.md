# 外部平台边界

| 外部平台 | 与清结算平台关系 | P0 契约 |
|---|---|---|
| 业务/订单/履约平台 | 提供标准履约完成事件 | `SettlementSourceEventCommand` |
| 账户账务平台 | 承接结算入账，返回账务请求号和流水号 | `SettlementAccountingPort` |
| 商户端/后台 BFF | 消费标准查询接口并自行适配页面口径 | 非 P0 设计主体 |
| 出款/提现平台 | 商户提现与出款处理 | 非 P0 主链路 |
| 退款/售后平台 | 后续逆向清分来源 | P0 不接入 |
| 对账平台 | 后续差错治理 | P1/P2 |

清结算平台不保存商户可用余额最终事实，不直接发起银行卡出款，不处理退款单生命周期。
