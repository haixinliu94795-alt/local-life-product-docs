# 清结算能力地图

<img src="../assets/diagrams/01_capability_map.png" width="920" alt="清结算能力地图">

## 设计目标

能力地图用于回答：清结算平台到底负责什么。P0 聚焦从标准业务事实到入账完成，不把提现、退款、完整对账、页面展示混进核心平台。

## 能力分层

| 层 | 能力 | P0 |
|---|---|---:|
| 接入层 | 标准业务事件接入、幂等处理 | 是 |
| 策略层 | 清分结算策略、规则快照 | 是 |
| 清分层 | 多参与方金额项生成 | 是 |
| 清算层 | 结算头寸、账期成熟、锁定 | 是 |
| 结算层 | 批次、结算单、明细 | 是 |
| 入账层 | 账务入账编排、UNKNOWN 补偿 | 是 |
| 查询层 | 标准查询能力 | 是 |
| 对账差错 | 漏结/重结/错结治理 | P1 |
