# 核心表设计

## 1. 表清单

| 表 | 上下文 | 一期 | 用途 |
|---|---|---:|---|
| cs_source_event | 业务接入 | 是 | 标准业务事件与幂等。 |
| cs_source_order_snapshot | 业务接入 | 是 | 订单/核销快照。 |
| cs_settlement_rule | 规则 | 是 | 规则主表。 |
| cs_rule_snapshot | 规则 | 是 | 清分使用的规则快照。 |
| cs_clearing_result | 清分 | 是 | 单笔清分结果。 |
| cs_clearing_result_item | 清分 | 是 | 多参与方金额项。 |
| cs_clearing_batch | 清算 | P1/可选 | 周期汇总/轧差批次，一期可先简化。 |
| cs_pending_settlement_item | 清算/待结算 | 是 | 待结算池。 |
| cs_settlement_batch | 结算 | 是 | 结算批次。 |
| cs_settlement_bill | 结算 | 是 | 结算单。 |
| cs_settlement_bill_item | 结算 | 是 | 结算明细。 |
| cs_settlement_voucher | 结算 | 是 | 结算凭证。 |
| cs_accounting_order | 账务编排 | 是 | 入账指令和状态。 |
| cs_operation_log | 审计 | 是 | 操作日志。 |
| cs_reconcile_task | 对账 | P1 | 对账任务。 |
| cs_reconcile_diff | 对账 | P1 | 对账差异。 |

## 2. 关键字段

### cs_clearing_result

| 字段 | 说明 |
|---|---|
| clearing_no | 清分单号 |
| source_event_no | 来源事件 |
| source_order_no | 业务订单号 |
| business_scene | 业务场景 |
| settlement_scene | 结算场景 |
| rule_snapshot_no | 规则快照 |
| gross_amount | 订单金额 |
| paid_amount | 实付金额 |
| merchant_amount | 商户应结金额 |
| platform_amount | 平台应收金额 |
| status | 清分状态 |

### cs_pending_settlement_item

| 字段 | 说明 |
|---|---|
| pending_item_no | 待结算项编号 |
| clearing_no | 清分单号 |
| participant_id | 结算主体 |
| amount | 待结算金额 |
| due_time | 可结算时间 |
| status | PENDING/MATURED/LOCKED/SETTLED/CANCELED/FROZEN |
| freeze_reason | 冻结原因 |
| lock_batch_no | 锁定批次 |

### cs_settlement_bill

| 字段 | 说明 |
|---|---|
| bill_no | 结算单号 |
| batch_no | 结算批次 |
| merchant_id | 商户 |
| business_scene | 业务场景 |
| settlement_scene | 结算场景 |
| period_start / period_end | 结算范围 |
| payable_amount | 应付金额 |
| status | 结算状态 |
| idempotent_key | 幂等键 |
| accounting_request_no | 账务请求号 |
| confirmed_by / confirmed_time | 确认人/时间 |
| settled_time | 入账成功时间 |
| fail_reason | 失败原因 |
| retry_count | 重试次数 |

## 3. 索引建议

| 表 | 索引 |
|---|---|
| cs_source_event | uk_source_event_no、uk_idempotent_key |
| cs_clearing_result | uk_scene_order_event、idx_source_order_no |
| cs_clearing_result_item | idx_clearing_no、idx_participant |
| cs_pending_settlement_item | uk_clearing_participant_amount_type、idx_merchant_status_due_time |
| cs_settlement_batch | uk_batch_no、idx_scene_merchant_period |
| cs_settlement_bill | uk_bill_no、uk_idempotent_key、idx_merchant_status_time |
| cs_settlement_bill_item | uk_pending_item_no_active、idx_bill_no |
| cs_accounting_order | uk_bill_no、uk_idempotent_key、idx_status_retry |
