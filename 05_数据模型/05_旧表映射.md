# 旧表映射

## 1. 旧表到新表映射

| 旧表 | 新表 | 说明 |
|---|---|---|
| `local_order_finance_detail` | `ccs_clearing_result`、`ccs_clearing_result_item` | P0 普通商品清分字段来源。 |
| `local_order_finance_detail.settle_status` | `ccs_pending_settlement_item.lifecycle_status` | 旧待结算标识迁移为生命周期状态。 |
| `local_settlement_record` | `ccs_settlement_bill` | 新单据中心替代旧结算记录。 |
| `local_settlement_record_order` | `ccs_settlement_bill_item` | 新结算明细替代旧关联表。 |
| `fund_account_flow` | 账户账务平台 | 清结算只保存流水号，不作为余额事实。 |

## 2. 双轨切换

```text
write_off_time < cutover_time  -> 旧链路收尾
write_off_time >= cutover_time -> 新清结算平台
```

## 3. 商户端展示

双轨期 BFF 合并新老数据，不允许前端直接拼表。
