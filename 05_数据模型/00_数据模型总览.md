# 数据模型总览

## 1. P0 表清单

| 表名 | 用途 |
|---|---|
| `ccs_source_event` | 标准来源事件 inbox。 |
| `ccs_clearing_result` | 清分结果主表。 |
| `ccs_clearing_result_item` | 清分明细表。 |
| `ccs_settlement_position` | 结算头寸表。 |
| `ccs_settlement_batch` | 结算批次表。 |
| `ccs_settlement_bill` | 结算单主表。 |
| `ccs_settlement_bill_item` | 结算单明细表。 |
| `ccs_accounting_posting_order` | 账务入账单表。 |
| `ccs_settlement_operation_log` | 结算操作日志表。 |

## 2. 建模原则

- 金额字段统一使用 `BIGINT`，单位分，人民币。
- 不设计 `currency / cur / currency_code` 字段。
- 不设计 `tenant_id` 字段。
- 不出现旧系统表名和字段映射。
- 不出现退款字段。
- 所有写入链路都有 `idempotent_key` 和 `request_hash`。
- 所有跨平台业务写入都包含 `caller_system / biz_domain / biz_no`。
