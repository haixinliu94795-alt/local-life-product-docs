# 产品验收口径

本文件只描述产品验收标准，不替代测试用例。

| 编号 | 验收口径 |
| --- | --- |
| A1 | 商品无未消费订单时，平台可完成单个下架，商品状态变为已下架。 |
| A2 | 商品存在未消费订单时，单个下架失败，并出现明确失败提示。 |
| A3 | 批量下架混合商品时，仅无未消费订单商品被处理；不可处理商品不被下架。 |
| A4 | 商品已下架后，不出现在用户可购买入口。 |
| A5 | 用户直接访问已下架商品时，不能继续购买，并出现已下架或不可购买提示。 |
| A6 | 店铺已下架后，用户访问店铺出现已下架提示，不再展示购买动作。 |
| A7 | 下架/封板不删除历史订单、交易快照、售后订单和结算记录。 |
| A8 | 商品上架若需审核，必须走审核通过后自动上架，不绕过审核。 |
| A9 | 删除商品时，有未消费订单不可删除；订单信息继续保留。 |
| A10 | 批量操作有二次确认，失败或过滤情况有明确反馈。 |
