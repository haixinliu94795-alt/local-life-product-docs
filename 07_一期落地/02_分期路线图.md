# 分期路线图

| 阶段 | 目标 | 主要能力 |
|---|---|---|
| 一期 | 当前普通商品清结算闭环 | 清分、待结算、后台手动结算、账务入账、商户查询 |
| P1 | 本地生活多业务接入 | 团餐、邀新、渠道、自动批次、基础对账、通用入账 |
| P2 | 多平台和复杂资金 | 按摩平台、技师/门店分润、退款冲正、调账补结、差错工作台 |
| P3 | 规模化运营 | 规则引擎、实时清分、风控冻结、财务报表、审计监控 |

## 一期里程碑

1. 建清结算模块与核心表；
2. 建本地生活普通商品防腐层；
3. 生成清分结果和待结算池；
4. 后台商家应付列表切新读模型；
5. 结算确认生成批次/单据/凭证；
6. 调账户账务入账并处理状态；
7. 商户端查询切新；
8. 最小对账和灰度上线。
