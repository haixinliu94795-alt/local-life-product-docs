# 字段设计说明

## 本章结论

字段设计必须服务领域语义、状态机和查询场景，不按旧表迁移字段命名。

| 字段类型 | 规则 |
|---|---|
| 编号字段 | 使用 `xxx_no`，例如 `event_no`、`clearing_no`、`position_no`。 |
| 金额字段 | 使用 `xxx_cent`，`BIGINT`，单位分。 |
| 状态字段 | 与状态机枚举一致。 |
| 业务归属 | `caller_system / biz_domain / biz_no / business_scene`。 |
| 幂等 | `idempotent_key / request_hash`。 |
| 时间 | `DATETIME(3)`。 |
| 逻辑删除 | `deleted TINYINT`。 |
| 乐观锁 | `version BIGINT`。 |
